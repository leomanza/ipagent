## Overview

## Getting Started

### Prerequisites

- Node.js v18 or higher
- npm v9 or higher

### Installation

```sh
# Clone the repository
git clone https://github.com/leomanza/ipagent.git

# Navigate to project directory
cd ipagent

# Install dependencies
bun install

# Start the development server
bun dev
```

### Building for Production

```sh
bun build
```
